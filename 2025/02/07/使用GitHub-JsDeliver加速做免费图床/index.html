使用GitHub+JsDelivr加速做免费图床

2025-02-07 服务部署 个人博客, 图床 0 评论

0.前言

在黑群晖本地搭建了lsky图床，很稳定，但是由于用的是ipv6访问，我的网络经常出问题，所以选择试一下GitHub做图床。也简单看了下国内免费的图床，多少有点不靠谱或者需要升级才能够用。

1. 开始布置

  • 先新建一个GitHub仓库，作为图片保存地

  • 获取GitHub token，给这个仓库全部权限即可。路径：头像-settings-developer settings- create new token

  • 下载picgo，配置GitHub图床，如图

  • 自定义域名是使用jsdelivr加速，还在试用中
