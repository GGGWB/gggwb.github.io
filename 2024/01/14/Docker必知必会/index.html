docker镜像原理

linux一切都是文件

  • 底层 bootloader+kernel=boot file system=bootfs 文件引导系统
  • 二层 root file system=rootfs 例如 /etc /sbin /root /dev Linux各种发行版都是基于rootfs
  • 容器创建了可读写的rootfs然后调用内核，所以容器小而快
